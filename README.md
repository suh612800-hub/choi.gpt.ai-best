# ArchiFlow AI Toolkit

## 소개

이 프로젝트는 건축 설계 과정에서 반복적으로 발생하는 도면 정리, 공간 분석, 발표 자료 작성을 보조하는 AI 기반 자동화 도구입니다.

건축 도면과 분석 데이터를 바탕으로 설계 내용을 정리하고, 체크리스트와 요약문을 생성하여 설계자가 더 효율적으로 작업할 수 있도록 돕는 것을 목표로 합니다.

## 주요 기능

* 건축 설계 자료를 바탕으로 분석 리포트 생성
* 교수님 피드백 및 도면 수정 사항을 체크리스트로 정리
* 공간 분석 결과를 발표용 문장으로 요약
* GPT, Claude, Codex 등 AI 도구 활용 결과 비교 정리

## 사용 방법

1. 저장소를 다운로드합니다.

```bash id="f3fbjj"
git clone https://github.com/your-username/archiflow-ai-toolkit.git
```

2. 프로젝트 폴더로 이동합니다.

```bash id="aw755r"
cd archiflow-ai-toolkit
```

3. 필요한 패키지를 설치합니다.

```bash id="gpr33z"
pip install -r requirements.txt
```

4. 예시 파일을 실행합니다.

```bash id="t1l0au"
python main.py
```

5. 생성된 결과 파일을 확인합니다.

```text id="0ftd0i"
outputs/
 ├── analysis_report.md
 └── checklist.md
```
